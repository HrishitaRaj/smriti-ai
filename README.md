# 🧠 Smriti AI
**Emotion-Aware Multimodal AI for Dementia Care**

Smriti AI is an emotion-aware, multimodal AI platform that captures and retrieves autobiographical memories using voice, text, and images, while integrating emotional context into memory recall.

Unlike traditional reminder systems, Smriti focuses on preserving identity, emotions, and personal experiences, enabling more meaningful and human-like interactions.

## Problem Statement

- Dementia patients suffer from memory loss and emotional instability
- Existing systems focus only on factual reminders, ignoring emotional context
- Caregivers lack tools to track emotional and cognitive changes over time

## Solution

Smriti AI bridges this gap by:
- Capturing multimodal memories (voice, text, images)
- Tagging memories with emotional context
- Enabling personalized conversational recall
- Providing caregiver dashboards with insights and alerts

## Key Features
**🧠 Multimodal Memory Capture**
- Voice journaling (speech-to-text)
- Text-based entries
- Image uploads for visual memory

**💬 Emotion-Aware AI Recall**
- LLM-based conversational memory retrieval
- Context-aware responses using semantic + emotional similarity

**📊 Emotion & Behavior Analysis**
- Sentiment and affect detection
- Longitudinal emotional trend tracking

**👩‍⚕️ Caregiver Dashboard**
- Mood and memory insights
- Weekly summaries
- Alerts for emotional or cognitive decline

**🔒 Privacy-First Design**
- Secure storage
- Controlled data access
  
## System Architecture

Smriti AI follows a **layered architecture**:

**1. User Interaction Layer**
- Patients: voice, text, images
- Caregivers: dashboard access

**2. Data Capture & Processing**
- Speech-to-text
- NLP (entity + emotion extraction)
- Computer vision

**3. Memory Encoding Layer**
- Semantic embeddings (vector storage)
- Emotion-tagged memory representation

**4. AI Recall & Emotion Analysis**
- LLM-based conversational engine
- Emotion detection models

**5. Insights & Recommendation Layer**
- Caregiver dashboard
- Smart nudges & alerts

## How It Works

1. User records memory via voice/text/image
2. System extracts:
    - People, events, emotions

3. Memory is stored as:
   - **Semantic embedding + emotional tag**

4. During recall:
   - AI retrieves contextually relevant memories
   - Generates **personalized conversational responses**

5. Caregivers receive:
   - Emotional trends
   - Alerts for unusual patterns
